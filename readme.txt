Document Web Scanning & Uploading with Dynamic Web TWAIN & Go
=======================================================================

The sample shows how to create a Web application for document imaging and uploading with Dynamic Web TWAIN & Go.


For more information, please refer to the Blog

http://www.codepool.biz/twain/scan-and-upload-documents-in-web-twain-and-go.html



How to Run



1. Install Go https://golang.org/dl/
2. Run 'go run server.go' 
4. Visit localhost:2014 in a browser
